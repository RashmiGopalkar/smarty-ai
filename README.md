# Smarty AI 📚✨

Smarty AI is a vibrant AI-powered study companion built with React and FastAPI.

## Features
- Smart study timer
- Adaptive quizzes
- Flashcards
- Homework mode
- Voice assistant
- XP rewards
- Character upgrades
- Calendar + timetable
- Friendly AI personality

## Tech Stack
### Frontend
- React
- Vite
- TailwindCSS

### Backend
- FastAPI
- Python
- SQLite/PostgreSQL

## Run Frontend
```bash
npm install
npm run dev
```

## Run Backend
```bash
uvicorn main:app --reload
```
